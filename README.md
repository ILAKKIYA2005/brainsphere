# 🧠 BrainSphere

BrainSphere is a modern interactive quiz and assessment platform built using HTML, CSS, and JavaScript.

It provides users with a smart quiz experience featuring:
- Real-time score tracking
- Countdown timer
- Multiple-choice questions
- Instant answer feedback
- Responsive UI design
- Quiz progress tracking
